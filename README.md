## 📌 Sobre o Projeto

O **Sammy-AI** é um projeto de Inteligência Artificial desenvolvido em formato de Notebook (`.ipynb`). O principal objetivo deste modelo é fornecer saídas limpas, precisas e estruturadas, evitando respostas redundantes ou incoerentes ("garbage outputs").

## 🚀 Funcionalidades

*   🧠 Processamento de linguagem natural otimizado.
*   📉 Filtro de saídas repetitivas ou irrelevantes.
*   📋 Execução direto no ambiente interativo do Jupyter/Google Colab.

## 🛠️ Como Executar

Para rodar o projeto localmente ou no Google Colab, siga os passos abaixo:

1. Baixe o arquivo da última versão na seção de **Releases**.
2. Abra o [Google Colab](https://colab.research.google.com/) ou o Jupyter Notebook na sua máquina.
3. Faça o upload do arquivo `Sammy0_2_1.ipynb`.
4. Execute as células de configuração e divirta-se!
